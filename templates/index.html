Система управления учетными записями

Добро пожаловать в систему управления учетными записями.

Открыть Swagger UI

Доступные API endpoints:

  • GET /accounts - Получить список всех учетных записей
  • POST /accounts - Создать новую учетную запись

Для тестирования API используйте Swagger UI или инструменты вроде Postman.
